## About  💁‍

Demo project of VK App with VK Connect

## How to start  💨
Run in bash what you prefer:

```bash
npm start
```
or
```bash
yarn start
```

## How to build  👷🏼‍

This will create production bundle, with treeshaking, uglify and all this modern fancy stuff

```bash
npm run build
```
or
```bash
yarn run build
```
